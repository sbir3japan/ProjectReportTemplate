# 0. 報告の鉄則（執筆前に必ず確認）

- 左側は「事実（Fact）」、右側は「意見（Opinion）」： 客観的な数字と、自分の解釈を混ぜない。
- 年間ゴールからの逆算： 「今期」の数字が「年間計画」の達成にどう寄与するかを常に意識する。
- So What?（だから何？）： 「〜をした」という行動報告ではなく、それによる「価値・影響」を書く。
